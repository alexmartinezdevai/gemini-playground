# First Prompt

## Prompt

Explícame qué es una API en dos frases.

## Response

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permite a diferentes aplicaciones comunicarse e interactuar entre sí. Actúa como un intermediario que facilita el intercambio de información y funcionalidades de forma estandarizada y segura.
